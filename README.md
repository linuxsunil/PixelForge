## ✨ Key Features

* 🔒 **Structural Privacy:** 39 out of 42 tools operate 100% offline using your local browser memory. Files never leave your device.
* 📄 **PDF Studio:** Programmatically merge multiple PDFs, extract specific page arrays, delete page ranges, reorder sheets, insert blank templates, inject dynamic watermarks, stamp page numbers, or crop structural layout boundaries.
* 🖼️ **Image Engine:** Precision resizing (by pixels, scale percentages, or bounding boxes), direct quality-controlled compression, dynamic coordinate cropping, flip operations, alpha-channel opacity adjustments, and a hidden metadata stripper that completely sanitizes files of private EXIF/GPS tracking logs.
* 🔀 **Conversion Suite:** Convert image queues to multi-page PDFs instantly. For complex PDF-to-Image rendering, the workspace safely loads a sandboxed Mozilla library (PDF.js) on the first click while keeping all processing local.
* 🗜️ **On-the-Fly Compression:** Leverages deep `pdf-lib` stream optimization sequences to compress loose PDF layout bytes without degrading text properties.
* 📦 **Automated Batching:** Integrated with client-side compression loops to pack multi-page extractions into instantly downloadable structured `.zip` packages.

## 📖 How to Use

### 1. Launch & Select
Open the single `Pixeltools.html` file in any modern browser (Chrome, Edge, Safari, Firefox) or access your hosted live link. Click on any specialized tool card from the dashboard categories.

### 2. Drop Your Files
Drag and drop your document target directly into the dynamic drop-zone area or browse your local file system. The application will instantly preview file metadata and frame structures in memory.

### 3. Tweak & Download
Adjust functional ranges, slider ratios, pixel dimensions, or text strings using the configuration inputs. Click the action button to let the client-side engine compute the changes and prompt an immediate browser download.

## 🧠 The "Local-First" Tech Stack

PixelForge functions entirely within the client-side run-time environment by mapping complex algorithmic operations to native browser APIs:
* **pdf-lib:** Reads binary components into raw memory buffers, handling programmatic generation, structural page duplication, and font embedding right inside browser memory arrays.
* **HTML5 Canvas & FileReader APIs:** Executes pixel-level matrix manipulation (including running a customized Convolution Kernel Filtering Matrix for high-frequency edge sharpening, applying radial gradient overlays for vignettes, and altering pixel index data for sepia tones).
* **JSZip:** Accumulates iterative file output data streams and compresses them into formatted ZIP packages completely within browser RAM.
